How does JavaScript Works?

JavaScript is a single-threaded language, it's mean javascript only can runs one task at a time. JavaScript is an interpreted language, it's not a compiled language. JavaScript runs the code from step by step. That means javascript runs the code from upper to lower. JavaScript also a client side scripting language. The client side term runs inside the browser. For runs the JavaScript code every browser provides their own javascript engine. Chrome runs the javascript code by V8 engine, Firefox runs the javascript code by SpiderMonkey engine, Edge runs the javascript code by Chakra engine and Safari runs the javascript code by JavaScript Core.
